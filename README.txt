# MacroMaster README

## Description
MyWebsite is a front-end web application that allows users to calculate their own required macros plan based on their goals and also to sign up for additional health and fitness services.

## Features
- User Registration.
- Personal macros calculator available online.
- User-Friendly Interface: The website offers a clean and intuitive user interface, making it easy to navigate and use.

## Technical Specifications
MacroMaster is built using the following technologies:

### Front-End
- HTML5 and CSS3

## Getting Started
To run MyWebsite locally on your machine, follow these steps:

1. Clone this repository: `git clone https://github.com/TheoBreaux/sba307-html-and-css`
2. Open your web browser and visit `http://localhost:3000` to access the website.
